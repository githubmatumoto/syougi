**立式

横Aマス、縦Bマスで、Aが奇数として

A>=3, B>=2

駒を２枚置いた時の左右同じ位置においた場合。つまり鏡像。

横側1列のみに駒を置いた時の鏡像数 = (A-1)/2

縦Bマスあるので

  B * (A-1)/2

Aのまんなかの縦のみに２枚置いた時の順列の数。まんなかに
置くとかならず鏡像

  B! / ((B-2)! 2!)


よって

 鏡像を除いた数 = (順列の数 + UNIQの数)/2

                = ((A*B)! / ((A*B-2)! * 2!) + B * (A-1)/2 +  B! / ((B-2)! 2!))/2


A=3, B=2で試し算
　　(6! / (4! 2!) + 2 * (2)/2 + 2! / (0! 2!))/2
    (15 + 2 + 1) / 2
    18 / 2 = 9 -> 正解

A=3, B=3で試し算

     (9! / (7! 2!) + 3 * 2/2 + 3! / (1! 2!))/2 = 21

A=9, B=9で試し算
     (81! / (79! 2!) + 9 * 8 / 2 + 9! / (7! 2!))/2 = 1656


  -> 前述の盤面からの計算と同じ！！
